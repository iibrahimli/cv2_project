The provided ZIP files must be extracted into the directory `data/` in the project root. The `data/` directory is expected to have the following structure:

```
data/
├── center_bias_density.npy
├── fixations/
│   ├── test
│   ├── validation
│   └── train
├── images/
│   ├── test
│   ├── validation
│   └── train
├── test_images.txt
├── train_fixations.txt
├── train_images.txt
├── train_small_images.txt
├── val_fixations.txt
└── val_images.txt
```
